# Getting Started

Get up and running with StackScope in minutes.

## Prerequisites

- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of Bitcoin and DeFi concepts

## Accessing StackScope

1. Visit the StackScope landing page
2. Click **Launch Dashboard** to access the main application
3. Navigate through the sidebar to explore different features

## Dashboard Overview

The StackScope dashboard consists of several key sections:

### Overview
The main dashboard providing a high-level view of the Stacks ecosystem including:
- Total Value Locked (TVL) across protocols
- Active protocols count
- 24-hour transaction volume
- Network health metrics

### Analytics
Deep-dive analytics tools for:
- Historical data analysis
- Protocol comparisons
- Trend identification
- Custom metric tracking

### sBTC Monitor
Real-time monitoring of sBTC including:
- Current supply and peg health
- Signer set metrics
- Peg stability indicators
- Historical peg data

### Network
Live network activity showing:
- Recent blocks
- Transaction feed
- Contract deployments
- Network statistics

### Protocols
Individual protocol pages with detailed metrics:
- Protocol-specific TVL
- Yield rates
- Active users
- Historical performance

## Navigation Tips

- Use the sidebar to quickly switch between sections
- The top bar provides quick stats and search functionality
- All metrics update in real-time
- Click on any chart for expanded views and detailed data
