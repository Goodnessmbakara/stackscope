StackScope provides real-time analytics and insights into the Bitcoin DeFi ecosystem, offering institutional-grade data for developers, researchers, and DeFi participants.

## What is StackScope?

StackScope is a comprehensive analytics platform that delivers:

- **Real-time On-chain Data**: Live tracking of transactions, blocks, and contract deployments
- **Protocol Performance**: TVL, yield metrics, and active user analytics across Stacks protocols
- **sBTC Health Monitoring**: Signer set decentralization and peg stability metrics
- **Developer API**: High-throughput REST and WebSocket feeds for custom applications

## Key Features

### Protocol Performance
Monitor and compare leading Stacks DeFi protocols with real-time metrics including Total Value Locked (TVL), yield data, and active user counts.

### sBTC Health Dashboard
Track the health of the sBTC peg with cryptographic confidence, including signer set decentralization metrics and peg stability indicators.

### Network Pulse
Visualize every block, transaction, and contract deployment on the Stacks network in real-time.

### Open Data API
Access high-throughput data feeds through REST endpoints and WebSocket connections for building proprietary models and applications.
